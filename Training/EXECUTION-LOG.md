- **主结果(v2,干净)**:构图层 Δ=1 段级 split 三制式 47.3/47.2/46.9%(n=4622);全 24 组合 47.3–51.2%;Δ≥2 ~50%;细节层 52.3%。**H3(单批同质)正式排除**。

### 2026-07-19 · 补测:patch 池化视图 + ≥3★ 全局段探针(事件级 split)

- **背景**:用户补充标注学——0-3★ 偏局部(临近相似选优)、3-5★ 偏全局(局部最优选出后事件内总体美学最优)。两段合法配对域不同:局部段=滑窗(已测),**全局段=事件全集**(比较当年就是全局发生的,宪法"禁跨段组对"约束的是 0-3★ 局部段,不约束此段)。
- **patch 池化补测**(`--with-patch`,局部段同框架):构图层 Δ=1 patch均值 49.2 / patch均+std 49.6%;细节层 52.4/51.0%——池化 patch 与 CLS 同样 chance。
- **全局段探针**(新增 `--min-rating 3 --pair-scope event` 模式):≥3★ 922 组、11 事件、32780 对(Δ=1 21201)。**事件级 split:全部 50.4/49.6/50.5、Δ=1 50.0/49.5/50.7、Δ≥2 51.2/49.4/50.4;3-4、4-5 边界 49.0-50.4%——全 chance**。**对级对照(乐观上界):Δ=1 79.3/79.1/84.0%**,vs 局部段对级仅 60-62%。
- **解读**:① 用户两段拆分的直觉被证实——≥3★ 全局段的特征-标签结构确实不同:**CLS 在事件内对高星美学排序有强可学性(79-84%),远高于局部段(60-62%)**;② 但事件级留出仍 50%——**绝对美学信号在 CLS 里存在(事件语境内),却不跨事件迁移**;③ 与宪法挑战 3/决策 3 预设一致:跨段/跨事件绝对尺度本就不能指望裸特征,靠锚定(评分偏移/b_seg/精修锚点)+ 泛化引入;④ 对级 79-84% 含泄漏(共享照片/事件内邻近对),是"事件内可学性"的上界证据,不是迁移能力证据。
- **结论**:H1 判定不变(无跨段/跨事件迁移),但问题锐化为 **特征弱 vs 标签事件特异**。梯 2 ViT-L 复测同一套三段探针(局部段/全局段/事件级)即成 2×2 分离:ViT-L 若事件内仍 ~8 成、跨事件仍 ~5 成 → 特征不是瓶颈,backbone 定格 ViT-S、直接进 M2 锚定路线;ViT-L 若跨事件显著上分 → 特征瓶颈坐实并获解,按梯子继续。
- **下一步**:spatial_probe 多样批复测(H2 收尾,附带)→ 梯 2(待用户批准)。

### 2026-07-19 · spatial_probe 多样批复测:H2 空间坍缩在干净数据上同样排除

- **背景**:用户质疑"池化证伪不了 patch"——成立(池化只证伪池化统计量);spatial_probe 用**保留 32×32 空间结构**的头做公正检验(空间金字塔分块池化 + 微型 CNN 直接吃网格)。
- **命令**:`Tools/.venv/Scripts/python.exe Training/probes/spatial_probe.py --db D:/PhotoDB/dataset/photos_dataset.db`(仓根)。
- **结果**(构图层 Δ=1 4622 对、段级 split,与 feature_probe 主口径同一批对、CLS 锚点 47.3% 核对一致):A 池化-线性 49.2±1.6 / B 池化-MLP 50.2±2.2 / C 空间金字塔 49.4±1.3 / D 微型CNN **50.7±3.7**;train acc 55.7–74.5%、pair 级泄漏 55.0–63.8%。最佳头 D 相对池化基线仅 +1.6pt(在 ±3.7% 噪声内),且 D 的 train acc 仅 55.7%——**连拟合训练集都勉强**。
- **结论**:H2(空间坍缩)在多样干净数据上同样排除——patch 32×32 网格里也没有浅层可及的排序信号;**瓶颈不是"池化丢信息",而是 ViT-S/16@518 特征本身不含可迁移的细粒度排序信息**。至此 ViT-S 全部特征形态判尽(CLS 三制式 / 池化 patch / 空间 patch / 事件条件化),决策 8 梯 2(ViT-L)是特征层唯一剩余杠杆(梯 4 LoRA 兜底)。
